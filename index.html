Habidoro



Habidoro is a productivity application that allows users to designate the total amount of time they want to work, splits that time into "pomodoros" and breaks of specified lengths, then tracks statistics and graphs of the users productive time. It was developed using Windows Presentation Foundation (WPF).

Scheduler



Scheduler is a productivity application that allows users to create personalized, reusable schedules. It was developed using Windows Presentation Foundation (WPF) and released on itch.io. I made this way back in 2016, before I knew how to use git, so unfortunately the source code is not available.

Archipelago


Archipelago is a roguelite tower defense game I released in January 2022 on Steam. It was developed using Unity and C#.

It ended up being a fairly large project, especially for a solo developer. I'm happy how I was able to maintain the speed of development throughout the whole project by writing decoupled, modular code. I was able to define all the "charms" - which are gameplay modifiers the player can find during the game - almost entirely within data, which allowed for rapid iteration.
