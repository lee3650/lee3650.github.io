Archipelago


Archipelago is a roguelite tower defense game I released in January 2022 on Steam. It was developed using Unity and C#.

It ended up being a fairly large project, especially for a solo developer. I'm happy how I was able to maintain the speed of development throughout the whole project by writing decoupled, modular code. I was able to define all the "charms" - which are gameplay modifiers the player can find during the game - almost entirely within data, which allowed for rapid iteration.
